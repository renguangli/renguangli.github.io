Java中有两种数据类型，基础数据类型和引用类型

基本数据类型分为byte、short、int、long、float、double、boolean、char

引用数据类型又分为类、接口、数组

## 8大基本数据类型

Java基本数据类型所占字节数和默认值

|基本数据类型|描述|字节|默认值|
|:-:|:-:|:-:|:-:|
|byte|字节型|1|0|
|short|短整型|2|0|
|int|整型|4|0|
|long|长整形|8|0|
|float|单精度浮点型|4|0.0|
|double|双精度浮点型|8|0.0|
|boolean|布尔型|	-|false|
|char|字符型|2|''|
