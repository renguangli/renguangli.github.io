Idea 常用快捷键 （Mac)

```
查找
全局查找
替换
复制行(直接复制到下一行)
删除行
剪切行
查找类、文件 ....
快速至行尾、行首
注释一行或着多行 //(注意这个要在语言是英文状态下)

快速回到上次修改的地方
代码向上下移动

出现生成get,set方法的界面
打开（进入）光标处的方法或者类
查看实现的类
查看谁调用了该方法

大小写转换
格式化代码。
优化导入的类和包
自动结束代码，行末自动添加分号; 

Command＋E     打开最近更改的代码
command＋O     打开文档（外部通用）

Command＋J     自动代码,比如for循环
```



```
Command＋C     复制
Command＋v     粘贴
command＋S     保存文件
Command＋F     查找
Ctrl＋Shift+F  全局查找
Command＋R     替换
Command＋D     复制行(直接复制到下一行)
Command＋Y     删除行
Command＋X     剪切行
Shift 双击     查找类、文件 ....
Command＋左右键 快速至行尾、行首
command+/     注释一行或着多行 //(注意这个要在语言是英文状态下)

Option＋Command＋左右键   快速回到上次修改的地方
Shift＋Command＋上下键 代码向上下移动

Ctrl＋Enter    出现生成get,set方法的界面
Command＋B     打开（进入）光标处的方法或者类
Command＋Option+B     查看实现的类
Ctrl＋Option+H    查看谁调用了该方法

Command＋Option＋L     格式化代码。
Command＋Option＋O     优化导入的类和包
Command＋Shift＋U      大小写转换
Command＋Shift＋Enter  自动结束代码，行末自动添加分号; 

Command＋E     打开最近更改的代码
command＋O     打开文档（外部通用）

Command＋J     自动代码,比如for循环
```
